# データフォルダ

このフォルダには、RAGシステムで使用する手順書ファイルを配置します。

## サンプルファイル

- `sample_manual.md`: サンプル手順書（デモ用）
- `incident_response.md`: インシデント対応手順書（デモ用）

## PDFファイルについて

**PDFファイルはGitで管理していません**（`.gitignore`で除外）

理由：
- リポジトリサイズの肥大化を防止
- Streamlit Cloud等でのcloneタイムアウトを回避
- 大容量ファイル（数十MB以上）は別管理が推奨

## 本番環境でのPDFファイル利用方法

### 方法1: ローカル環境での利用
ローカルで開発する場合は、このフォルダに直接PDFファイルを配置してください。

```bash
# PDFファイルを配置
cp /path/to/your/manual.pdf data/

# アプリを起動
streamlit run main.py
```

### 方法2: アプリ内アップロード機能を利用
アプリのサイドバーから「📤 手順書アップロード」機能を使って、ブラウザから直接PDFファイルをアップロードできます。

### 方法3: 外部ストレージ連携（将来実装）
S3、Google Cloud Storage等の外部ストレージからファイルを読み込む方法も検討できます。

## 推奨ファイル形式

- **Markdown (`.md`)**: 推奨。見出し構造がそのまま使える
- **テキスト (`.txt`)**: UTF-8エンコーディング推奨
- **PDF (`.pdf`)**: テキスト抽出可能なPDF（スキャン画像のみは非対応）

## 注意事項

- パスワード保護されたPDFは読み込めません
- 機密情報を含むファイルをアップロードしないでください
- ログに記録されるため、個人情報等の取り扱いに注意してください
